# Pref4git
### Quickely setup git

## Support enviroments
- macOS
- Linux
- Windows

## Usage
### 1) Initial git setup
If you install new OS, try this:
#### macOS and Linux
```bash
sh initial-git.sh
```
#### Windows
```powershell
initial-git.bat
```
### 2) Git configure
If you only setup git basic configure, try this:
#### macOS and Linux
```bash
sh git-conf.sh
```
#### Windows
```powershell
git-conf.bat
```
- Branch master to main
- Add user name and email
### 3) Git global ignore
If you want only setup git global ingnore, try this:
#### macOS and Linux
```bash
sh git-ignore.sh
```
#### Windows
```powershell
git-ignore.bat
```
